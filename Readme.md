# Cross-correlation for ambient noise data


## Introduction

1. Create a CONDA environment with the following packages:
    - obspy
    - pyasdf
    - mpi4py

  ```bash
  conda create -n cccn obspy pyasdf mpi4py -c conda-forge
  ```

2. Activate the environment and install the package

  ```bash
  conda activate cccn
  pip install .
  ```
